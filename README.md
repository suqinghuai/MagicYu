# MagicYu 魔鱼 - 闲鱼自动回复系统

## 项目简介

MagicYu（魔鱼）是一个基于Python开发的闲鱼自动回复系统，能够自动检测新消息并使用AI生成回复内容，帮助卖家提高回复效率，提升客户体验。

## 功能特点

- **自动检测新消息**：通过像素颜色检测技术，实时监控闲鱼聊天界面的新消息提示
- **双模型链路**：识图模型专门提取聊天截图文本，对话模型专门生成客服回复
- **商品识别**：通过像素颜色检测识别当前显示的商品，为AI提供针对性的回复提示
- **自动操作**：模拟鼠标点击和键盘输入，实现全自动化回复流程
- **配置灵活**：通过配置文件可自定义各种参数，适应不同的屏幕尺寸和布局
- **日志记录**：详细记录系统运行状态和错误信息，便于问题排查

## 安装步骤

### 1. 环境要求

- Python 3.7+
- Windows 操作系统（支持屏幕截图和鼠标操作）

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置文件设置

编辑 `config.ini` 文件，根据您的屏幕布局和需求进行配置：

- **设置**：调整检查间隔和点击间隔
- **新消息**：设置新消息提示的像素坐标和颜色
- **商品**：配置不同商品的像素坐标、颜色和对应提示词
- **坐标**：设置对话框、输入框、发送按钮等界面元素的坐标
- **API**：配置AI模型和API密钥

### 4. 运行程序

```bash
python main.py
```

## 工作原理

1. **监控新消息**：程序定期检查指定坐标的像素颜色，判断是否有新消息
2. **点击对话框**：当检测到新消息时，自动点击对话框打开聊天界面
3. **识别商品**：通过检测商品区域的像素颜色，识别当前显示的商品
4. **截图分析**：截取聊天区域的截图，用于AI分析
5. **识图模型提取文本**：调用视觉模型识别聊天截图内容
6. **对话模型生成回复**：将识别文本和商品提示词发送给对话模型生成回复
7. **自动回复**：将AI生成的回复内容粘贴到输入框并发送
8. **回到初始状态**：点击空白区域，回到消息列表界面，准备处理下一条消息

## 配置说明

### 1. 新消息检测配置

在 `config.ini` 的 `[新消息]` 部分，设置新消息提示图标的坐标和颜色：

```ini
[新消息]
横坐标 = 680
纵坐标 = 496
红色值 = 255
绿色值 = 77
蓝色值 = 79
```

### 2. 商品检测配置

在 `config.ini` 中为每个商品设置检测坐标、颜色和提示词：

```ini
[商品1]
横坐标 = 200
纵坐标 = 200
红色值 = 0
绿色值 = 255
蓝色值 = 0
提示词 = 你好，这是商品1的回复提示词。
```

### 3. 界面坐标配置

在 `config.ini` 的 `[坐标]` 部分，设置界面各元素的坐标：

```ini
[坐标]
对话框横坐标 = 805
对话框纵坐标 = 520
输入框横坐标 = 1045
输入框纵坐标 = 900
发送按钮横坐标 = 1870
发送按钮纵坐标 = 959
空白区域横坐标 = 797
空白区域纵坐标 = 322
聊天区域横坐标 = 971
聊天区域纵坐标 = 356
聊天区域宽度 = 941
聊天区域高度 = 474
```

### 4. API配置

在 `config.ini` 中可以使用以下三种配置：

- `[API]`：通用兜底配置（不区分识图和对话时可只配置这一段）
- `[识图API]`：截图识别模型配置
- `[对话API]`：文本回复模型配置

示例：

```ini
[API]
模型 = Qwen/Qwen3-VL-8B-Instruct
API密钥 = your_api_key_here
基础URL = https://api-inference.modelscope.cn/v1/

[识图API]
模型 = Qwen/Qwen3-VL-8B-Instruct
API密钥 = your_api_key_here
基础URL = https://api-inference.modelscope.cn/v1/

[对话API]
模型 = Qwen/Qwen3-8B
API密钥 = your_api_key_here
基础URL = https://api-inference.modelscope.cn/v1/
```

## 使用注意事项

1. **屏幕分辨率**：确保您的屏幕分辨率与配置文件中的坐标设置匹配
2. **闲鱼界面**：保持闲鱼界面在前台运行，避免被其他窗口遮挡
3. **网络连接**：确保网络连接正常，以便调用AI API
4. **API密钥**：妥善保管您的API密钥，避免泄露
5. **权限设置**：确保程序有足够的权限进行屏幕截图和鼠标操作
6. **定期检查**：定期检查日志文件，了解系统运行状态

## 常见问题

### 1. 程序无法检测到新消息

- 检查 `config.ini` 中 `[新消息]` 部分的坐标和颜色设置是否正确
- 确保闲鱼界面在前台显示，且新消息提示图标清晰可见

### 2. AI回复内容不准确

- 检查 `config.ini` 中商品的提示词设置是否合理
- 确保聊天区域截图包含完整的对话内容
- 考虑调整AI模型或提示词以获得更好的回复质量

### 3. 程序运行时出错

- 检查日志文件了解具体错误信息
- 确保所有依赖包已正确安装
- 检查 `config.ini` 中的配置是否正确

### 4. 自动操作失败

- 检查 `config.ini` 中各界面元素的坐标设置是否正确
- 确保闲鱼界面没有被其他窗口遮挡
- 考虑调整 `点击间隔` 设置，给系统更多响应时间
